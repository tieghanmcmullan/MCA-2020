Task 1: Generating a Similarity Matrix

For this part of the task, I have used different tracks from the last few weeks. In the similarity matrix:
Tracks 0, 1, 2 and 3 are of the classical genre.
Tracks 4, 5, 6 and 7 are of the rock genre.
Tracks 7, 8 and 9 are my tracks which are all of the pop genre.
The matrix reflects these genres as the rock tracks (4, 5 & 6) and the pop tracks (7, 8 and 9) and more similar to eachother whilst the classical tracks look quite different. This could be because these tracks have similar tempo and notes played and sometimes these genres can coincide whereas classical is very different.
Furthermore, my track number 7 (pop) is quite similar to track 4 which is rock, but it is unlike track 0, 1, 2 and especially 3 which are all classical tracks. My track number 8 is especially similar to track 4 again but especially unlike track 3 again. Finally, my track number 9 is quite similar to track 4, 5 and 6 and again most unlike track 3.

Task 2: Transcription

In Task 2, I used the file I created in Week 2 of "Can't Help Falling In Love" by Elvis. I did this by:

  • Exporting the MuseScore (.msc) file into a wav file.
  • Importing this wav file into SonicVisualiser where I transformed the file into a piano roll
  • I then exported this piano roll into a MIDI file
  • I opened this MIDI file in MuseScore and then exported it as an image file
I have included 2 images below which show my original transcribed piece and the updated file I created this week.

The two files are very clearly different. First of all, the new file has an F sharp key signature which was not in the original so this is not ideal. The tempo and time signature has remained the same in both files which is better, however by the looks of it, every bar has changed and as a result the new file sounds very different from the original.

The notes in bars 14 and 15 sound very different - the new file has these notes really quite high when as in the original they should be quite low to go with the tune we all know. There are definitely a lot more notes in the new version compared to the original and this makes the sound much more layered.

My favourite part of the melody is in bars 33/34 to around 43/44 - I feel the new file has definitely enhanced the original at this point in the song with more notes it creates more depth even though it has changed it slightly.

Overall, this would suggest that the automatic transcription has changed quite a lot about the track.
Perhaps, it would be more accurate and reliable for a human being to transcribe music with computer aided systems rather than the computer doing it all.
